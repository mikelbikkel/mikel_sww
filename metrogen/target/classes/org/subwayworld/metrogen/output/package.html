Output processing.

This packages contains classes that generate output from the MetroNetwork structure.

Error handling.

Module name for errorcodes: OUTPUT. Error codes:
  • OUTPUT-00001 = Unknown city_id/city_code combination.
  • OUTPUT-00002 = "Cannot create output file.".
  • OUTPUT-00003 = "Connection error.".
  • OUTPUT-00004 = "SQL error.".
OUTPUT-00001 city_id city_code.
OUTPUT-00002 filename.
OUTPUT-00003 RSConnectionException.
OUTPUT-00004 SQLException.
